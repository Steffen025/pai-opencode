### Key Technical Discovery

While debugging, we discovered an important OpenCode API quirk that might help others building plugins:

```typescript
// In tool.execute.before, args are in OUTPUT, not input!
"tool.execute.before": async (input, output) => {
  const command = output.args.command; // ‚Üê Correct!
  // NOT: input.args.command  ‚Üê Wrong!
}
```

This is now documented in our `TEST-RESULTS-v0.7.md` for future reference.
